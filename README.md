# Laravel + inertia + Vue 3

## Introduction

> Example Laravel + inertia + Vue 3 project

- Laravel 8
- Laravel-mix 6 (webpack 5)
- Vue 3
- Bootstrap 5
- Ziggy Routes

## Installation

> git clone git@github.com:smadrom/laravel-inertia-vue3.git

> cd laravel-inertia-vue3

> php artisan serve

> **npm run dev** or **npm run watch**
